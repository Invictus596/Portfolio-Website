# About This Site

This site is inspired by macOS [Big Sur](https://www.apple.com/in/macos/big-sur/) and [Catalina](https://www.apple.com/bw/macos/catalina/), developed using [React](https://reactjs.org/), [Zustand](https://zustand-demo.pmnd.rs/) and [UnoCSS](https://uno.antfu.me/), and hosted on [Vercel](https://vercel.com/).

The original project was created by [Renovamen](https://github.com/Renovamen/playground-macos). I've personalized it to showcase my work and profile.

The source code is hosted [here](https://github.com/Invictus596/Portfolio-Website).
